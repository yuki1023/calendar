## 使用技術
- 開発環境  
　   - Xcode,swift  
- DB  
　   - mBaaS,UserDefaults  
- カレンダー  
　   - FSCalendar  
- 画面遷移時の非同期処理への対応  
　   - PKHUD  


## 主な機能  
- カレンダーの日付をタップするとその日付の情報を取得  
- 日付毎にスケジュールを投稿
- チェックボックスで達成と未達成を記録
- その日の達成度を面白みを込めながら見れる
- 匿名ログイン
